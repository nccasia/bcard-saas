# Description

Bcard-saas is a web project that creates personalized business cards for employees of the NCC company. The code is built using Prisma ORM, NextAuth, Next.js , ReactJS,...

# Getting Started

## Installation

```bash
$ npm install
```

## Run

```bash
$ npm run dev
```

## Test

```bash
$ npm run build
```

## Edit Data

```bash
$ npx prisma
```
